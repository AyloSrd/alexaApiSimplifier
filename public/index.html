Alexa Rank API Simplifier

Welcome Alexa Rank API Simplifier, the easiest way to make GET requests the alexa ranking from your Salesforce API.

  1. Create an AWS awis account and get your API key.
  2. Make sure https://alexa-api-simplifier.herokuapp.com/ is included in you CSP Trusted Sites list on Saleseforce (how to here)
  3. Choose the API COmponent of your choice. The url for the GET request must be structured as follows : https://alexa-api-simplifier.herokuapp.com/api/YOUR AWIS API KEY/THE DOMAIN YOU ARE INSPECTING
